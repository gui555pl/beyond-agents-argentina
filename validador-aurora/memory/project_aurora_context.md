A **Beyond** é a venture builder do Extreme Group — "fábrica de startups". A **Aurora** é a BU responsável pela fase inicial: pegar uma hipótese e validar se tem potencial de escala antes de transferir para Venture Studio (escala) ou Volund (dev).

Hoje o Comitê de Inovação Aurora aplica o **scorecard do Playbook de Seleção** manualmente na Fase 1 (Screening), antes de qualquer investimento. Esse trabalho consome semanas de pessoa por hipótese — o que limita o número de apostas simultâneas que o grupo carrega.

**Why:** o Validador Aurora automatiza esse screening para reduzir o custo de validação de "semanas + pessoas" para "minutos + tokens", liberando capacidade humana para a decisão final do Comitê.

**How to apply:**
- Toda avaliação deve seguir exatamente o scorecard do Playbook (16 critérios, pesos, faixas de recomendação).
- Quando houver ambiguidade nas regras, consultar o time Aurora antes de inferir.
- O Validador **alimenta** o Comitê, não substitui — output é sinal, não comando.
- O agente é parte de um pipeline maior (Orquestrador-Workers) — outputs seguem schema rígido para integração.

**Hackathon (referência temporal):** Beyond Agents, 22-23 mai 2026, Recife. Pipeline completo: Orquestrador + Validador Aurora + Buscador + Criador de Assets + Deploy Vercel + Gestor de Tráfego + Miro Fish + Análise de Performance.
